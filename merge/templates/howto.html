How to use

  • Make sure you followed the installation instructions in the Github
  • By default the index_name and the elasticsearch instance correspond as in the installation
  • Choose one of the 3 available modes on the right side: url,text, file
  • Fill the textfield with the corresponding information: url >> link; text or file >> plaintext
  • Press 'submit'

In case of errors

  • Press 'status'
  • Ensure elasticsearch instance you tried to use was found.
    • If False: ensure elasticsearch is running in the background.

  • Ensure index_name you tried to use was found.
    • If False: if the elasticsearch index was found you can recreate the index
    • If True but the search is not working: Press 'clear' and afterwards recreate the index. May take several minutes (page refreshes on finish)

  • Ensure model for ranking was found.
    • If False: Press 'download'. May take some minutes depending on download speed (page refreshes on finish)

Miscellaneous

  • We recommand using file or text mode over url, to obtain better text bodies.
  • Url mode can possibly be broken given different structures of the source websites (e.g. nytimes.com).
  • In url mode don't add anything behind the url in the textarea (don't hit Enter on keyboard).
